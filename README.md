# Auto-Pin Domains 

Chrome extension that auto-pins tabs from specified domains.  
Built to save your tabs from chaos. 

:Features
- Auto-pins tabs from your saved domain list
- Toggle enable/disable
- Easy UI to add/remove domains

:Credits
Made with code, coffee, and existential dread  
(@MarlGian + AI assist)

:Setup
1. Clone or download this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the extension folder
5. Enjoy eternal pinned-tab peace
